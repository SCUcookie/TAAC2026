# Platform Feedback Log

Put each platform run into a separate markdown file in this directory.

Recommended filename format:
- `2026-05-06_run01.md`
- `2026-05-06_eval01.md`

Record at least:
- submission directory used
- exact date/time
- training command or platform setting
- whether training finished
- best checkpoint path
- validation / leaderboard metric
- runtime or memory errors
- OOM / timeout / file-not-found / import issues
- any platform-specific warning text

Use the template file in this directory for each new run.
