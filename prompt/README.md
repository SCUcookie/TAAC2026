# Prompt Hub

This folder stores the working prompt set for the whole project lifecycle.

Start with `prompt/codex_workflow.md`. That file is the single entry point for the workflow.

## Files

- `codex_workflow.md`: master workflow prompt and operating rules
- `overview.md`: end-to-end workflow summary
- `file_structure.md`: current repository layout and folder roles
- `strategy.md`: modeling and experiment strategy
- `plan.md`: active plan with next actions and promotion rules

`codex_workflow.md` is the only file that needs to be read first. The other files are supporting notes.

Update these files whenever the experiment state changes.
