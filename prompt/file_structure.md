# File Structure

## Root

- `main.py`: primary training entry point
- `infer.py`: primary inference entry point
- `dataset.py`: shared dataset logic
- `model.py`: shared model definition
- `run.sh`: training launcher
- `run_taac2026.sh`: TAAC2026 launcher

## Baseline

- `baseline/training/`: training-only copy of the baseline code
- `baseline/evaluation/`: evaluation-only copy of the baseline code
- `baseline/README.md`: split overview

## Planning

- `prompt/`: live process notes, ideas, and plan
- `feedbacks/records/`: chronological feedback log with acc and run notes

## Reference Material

- `docs/`: competition notes, planning notes, and templates
- `champion_plan/`: reference solutions
- `original/`: original source snapshot
